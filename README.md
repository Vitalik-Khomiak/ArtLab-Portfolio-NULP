# Цифровий простір учителя математики

Статичний сайт-портфоліо, створений за вимогами практичних робіт:

- `Пр 1-2-3. Розробка цифрового портфоліо вчителя математики`
- `Пр 4-5. Візуалізація наукового дослідження`
- `Пр 6-7. Інновації в математиці`

## Як відкрити локально

Відкрийте файл `index.html` у браузері. Сайт не потребує Node.js, npm або сервера.

## Як опублікувати через GitHub Pages

1. Завантажте в репозиторій вміст папки `Сайт`.
2. У GitHub відкрийте `Settings` → `Pages`.
3. Оберіть гілку `main` і папку `/root`, якщо файли сайту лежать у корені репозиторію.
4. Збережіть налаштування та дочекайтеся посилання GitHub Pages.

## Склад сайту

- `index.html` — головна сторінка-візитка.
- `module1.html` — цифрова дидактика, сертифікати, есе і рефлексія.
- `module2.html` — візуалізація наукового дослідження.
- `mini-project.html` — підсторінка Модуля 2 з міні-проєктом.
- `module3.html` — міні-проєкт з інновацій у математиці.
